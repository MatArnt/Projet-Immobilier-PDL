# Projet DVF - Pays de la Loire

Projet d'économétrie spatiale sur l'évolution des prix immobiliers (Maisons et Appartements) dans la région Pays de la Loire.

## Contenu du dépôt
- `code_projet.R` : Le script R contenant le nettoyage des données, les modèles hédoniques à effets fixes, et la génération des cartes/heatmaps.
- `rapport_final.pdf` : Le rapport complet contenant l'analyse, les graphiques générés et nos interprétations économiques.
